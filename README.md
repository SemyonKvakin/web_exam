Квакин Семён Дмитриевич, группа 241-326.

## Стек

Flask, SQLAlchemy, Flask-Login, Flask-Migrate, Bootstrap 5, EasyMDE, Bleach, Markdown.

## Запуск

```
pip install -r requirements.txt
```

Настройте `config.py` (строка подключения к БД, `UPLOAD_FOLDER`).

Создание таблиц и наполнение начальными данными (роли, статусы рецензий, жанры, тестовые пользователи):

```
python seed.py
```

Либо через миграции:

```
flask db init
flask db migrate
flask db upgrade
```

Запуск:

```
flask run
```

## Реализованная функциональность

- Реестр книг с пагинацией (не более 10 на странице), сортировка по году.
- Роли: администратор, модератор, пользователь, разграничение доступа.
- Просмотр книги, описание из Markdown в HTML, обложка.
- Добавление/редактирование книги через общий макрос формы; обложка только при создании.
- Удаление книги через модальное окно, каскадное удаление связанных записей и файла обложки.
- MD5-дедупликация обложек.
- Санитайзер (Bleach) для пользовательского ввода.
- Создание рецензий со статусом «на рассмотрении» по умолчанию.
- На странице книги показываются только одобренные рецензии.
- «Мои рецензии» для пользователей.
- «Модерация рецензий» для модераторов: список на рассмотрении с пагинацией, страница рассмотрения с кнопками «Одобрить» / «Отклонить».
