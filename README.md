## Запуск

```
pip install -r requirements.txt
```


Создание таблиц и наполнение начальными данными (роли, статусы рецензий, жанры, тестовые пользователи):

```
python seed.py
```

Либо через миграции:

```
flask db init
flask db migrate
flask db upgrade
```

Запуск:

```
flask run
```
